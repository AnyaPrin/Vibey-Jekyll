Markdown チートシート

基本的なMarkdownの書式をまとめたものです。

見出し (Headers)

# の数で階層を表現します。

# 見出し1
## 見出し2
### 見出し3
#### 見出し4

テキストの装飾 (Text Emphasis)

太字: **太字** または __太字__

斜体: *斜体* または _斜体_

太字と斜体: ***太字と斜体***

取り消し線: ~~取り消し線~~

リスト (Lists)

順序なしリスト (Unordered List)

*、-、+ のいずれかを使い、スペースを入れます。

  • 項目1
  • 項目2
    • 項目2-1
    • 項目2-2
* 項目1
* 項目2
  * 項目2-1
  * 項目2-2

順序付きリスト (Ordered List)

数字と . を使います。

  1. 項目A
  2. 項目B
  3. 項目C
1. 項目A
2. 項目B
3. 項目C

リンクと画像 (Links and Images)

リンク: [表示テキスト](URL)

例：Google

画像: ![代替テキスト](画像URL)

例：

コード (Code)

インラインコード

文章中にコードを埋め込むには、`で囲みます。

例：これは console.log() のサンプルです。

コードブロック

複数行のコードを記述するには、` ` `で囲みます。言語名を指定すると、シンタックスハイライトが有効になります。

function hello() {
  console.log("Hello, World!");
}

表 (Tables)

| (パイプ) と - (ハイフン) を使います。

ヘッダー1 ヘッダー2
セル1 セル2
セル3 セル4 
| ヘッダー1 | ヘッダー2 |
| :--- | :---: |
| セル1 | セル2 |
| セル3 | セル4 |

引用 (Blockquotes)

> を使います。

これは引用です。

別の引用行。

> これは引用です。
> 別の引用行。
